# Aletheia

Aletheia helps women uncover what’s really happening in their bodies and minds during perimenopause and menopause—turning confusion into clarity, and symptoms into understanding.

## Features
- **Symptom Tracking**: Log daily symptoms with granular severity ratings.
- **Pattern Recognition**: Visualize history with calendars and charts.
- **Personalized Insights**: Educational content tailored to your stage.
- **Community**: Connect with others through online and in-person events.

## Getting Started
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
